# サーバー再起動の確認方法

## 重要なポイント

コードを変更した場合、**必ずサーバーを再起動する必要があります**。

## 再起動手順

### ステップ1: 現在のサーバーを停止

バックエンドサーバーを実行しているターミナルで：

1. **`Ctrl + C` を押す**
2. サーバーが停止するまで待つ
3. プロンプト（`PS ...>`）が表示されれば停止完了

### ステップ2: サーバーを再起動

同じターミナルで：

```powershell
npm run server
```

### ステップ3: 起動確認

以下のメッセージが表示されれば成功：

```
✅ Dify API プロキシサーバーを起動しました
   API Base URL: https://api.dify.ai/v1
🚀 サーバーが http://localhost:3001 で起動しました
📝 フロントエンドから http://localhost:5173 にアクセスしてください
```

### ステップ4: ブラウザをリロード

ブラウザで `http://localhost:5173` をリロード（F5）してください。

## トラブルシューティング

### エラー: "Port 3001 is already in use"

→ 別のターミナルでサーバーが起動している可能性があります。
1. すべてのターミナルを確認
2. サーバーを実行しているターミナルを見つける
3. そのターミナルで `Ctrl + C` を押して停止
4. 再度 `npm run server` を実行

### サーバーが起動しない

→ エラーメッセージを確認してください。よくある原因：
- `.env`ファイルが正しく設定されていない
- 必要なパッケージがインストールされていない（`npm install`を実行）

### それでもエラーが出る場合

→ ターミナルのエラーメッセージを確認してください。
