# PATH環境変数の修正方法

## 問題

PATH環境変数に `C:\Program Files\nodejs\` のように最後に `\`（バックスラッシュ）が付いている場合、問題が発生する可能性があります。

## 解決方法

### ステップ1: 環境変数の設定画面を開く

1. Windowsキー + R を押す
2. `sysdm.cpl` と入力してEnter
3. 「詳細設定」タブ → 「環境変数」をクリック
4. 「システム環境変数」の「Path」を選択 → 「編集」をクリック

### ステップ2: nodejsのパスを修正

1. リストから `C:\Program Files\nodejs\` を探す
2. 選択して「編集」をクリック
3. 最後の `\` を削除して、`C:\Program Files\nodejs` にする
4. 「OK」をクリック

または、削除して新規追加：

1. `C:\Program Files\nodejs\` を選択
2. 「削除」をクリック
3. 「新規」をクリック
4. `C:\Program Files\nodejs` と入力（最後の `\` は付けない）
5. 「OK」をクリック

### ステップ3: 適用

1. すべてのウィンドウで「OK」をクリック
2. **すべてのターミナルを閉じる**
3. **新しいターミナル（PowerShell）を開く**
4. 以下を実行：
   ```powershell
   node --version
   npm --version
   ```

## 正しい形式

- ✅ `C:\Program Files\nodejs` （推奨）
- ✅ `C:\Program Files\nodejs\` （動作するが、最後の `\` は不要）

## 注意

- パスの最後に `\` を付けても動作することが多いですが、付けない方が推奨されます
- 修正後は必ずターミナルを再起動してください
