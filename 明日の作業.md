## 明日やること

### 1. Node.jsのインストール確認

**確認手順:**
1. 新しいターミナル（PowerShell）を開く
2. 以下のコマンドを実行：
   ```powershell
   node --version
   npm --version
   ```
3. バージョン番号が表示されればOK

**もしインストールされていない場合:**
- `SETUP_NODEJS.md` を参照してインストール
- または `README.md` の「前提条件: Node.jsのインストール」セクションを参照

### 2. プロジェクトのセットアップ

Node.jsのインストール確認後：

```powershell
cd project-bolt-sb1-pcvnpemp/project
npm install
```

### 3. 環境変数の設定

1. `.env`ファイルを作成：
   ```powershell
   copy env.example.txt .env
   ```

2. `.env`ファイルを開いて、以下を設定：
   - `VITE_DIFY_API_KEY`: Dify API Key
   - `VITE_DIFY_APP_ID`: Dify App ID
   - `VITE_DIFY_API_BASE_URL`: `https://api.dify.ai/v1`（デフォルト）

### 4. 開発サーバーの起動

```powershell
npm run dev
```

ブラウザで `http://localhost:5173` にアクセスして動作確認
